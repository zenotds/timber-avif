# Timber AVIF Converter (Plugin)

This plugin bundles the v3 converter so you can enable AVIF/WebP for Timber without touching the theme. See the root docs for full details:

- [README-V4.0.md](../README-V4.0.md) for plugin usage
- [README-V3.0.md](../README-V3.0.md) for the shared converter API

## Quick start
1. Copy this folder to `/wp-content/plugins/` and activate it.
2. Import the macro from `timber-avif/macros/picture.twig` if desired.
